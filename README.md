# 📒 Agenda de Contatos

Projeto desenvolvido em Python com o objetivo de praticar conceitos fundamentais de programação, como modularização, manipulação de listas, dicionários, funções e persistência de dados.

## 🚀 Funcionalidades

* Adicionar contatos
* Listar contatos cadastrados
* Buscar contatos pelo nome
* Editar informações de um contato
* Remover contatos
* Armazenamento dos dados em arquivo JSON

## 🛠️ Tecnologias Utilizadas

* Python 3
* JSON
* GitHub

## 📂 Estrutura do Projeto

```text
agenda-contatos-python/
│
├── main.py
├── contact.py
├── files.py
│
├── contacts.json
│
├── README.md
└── .gitignore
```

## ▶️ Como Executar

1. Clone o repositório:

```bash
git clone URL_DO_REPOSITORIO
```

2. Acesse a pasta do projeto:

```bash
cd agenda-contatos-python
```

3. Execute o programa:

```bash
python main.py
```
